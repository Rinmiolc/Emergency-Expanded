# Emergency Expanded (急救与医疗扩展) - RimWorld 1.6 Hardcore Medical Mod

`Emergency Expanded` 是一款专为 **RimWorld 1.6** 设计的超硬核、高度拟真且极具操作博弈空间的医疗与战地急救扩展 Mod。

本 Mod 的设计初衷旨在重塑 RimWorld 相对单一的伤病体系，围绕战斗产生的急性危症（如低血容量休克、全身缺氧、酸碱崩盘与冠脉阻塞）建立起一套**逻辑自洽、病理科学且高度拟真**的临床医学级急症维生系统。在兼顾原版可玩性与多 Mod 兼容性的同时，极大拔高战斗伤亡的代价与生死抢救的临场刺激感！

---

## 核心机制与目前已实现功能

目前，本 Mod 已经实现并稳定部署了以下九大核心硬核医学与视觉对齐系统：

### 1. 嵌入式液晶心电图监视仪 (ECG Vital Monitor)
在玩家当前选中的血肉生物控制面板中，新增了一个与原版 Gizmo 按钮在**物理高度、斜边切角和视觉边框上 100% 严丝合缝、完全对齐**的液晶黑色液晶体征监护仪：
* **动态心率双重代偿模型**：
  * **急性流血交感亢奋**：融入全身总流血率 `BleedRateTotal`。当伤员刚受创开始大量暴血时，即使失血 Hediff 尚未累积，交感神经也会立刻被点燃，心率动态飙升 `+20 到 +60 bpm`，完美还原临床急性出血表现。
  * **累积失血高原期**：失血严重度持续上升时，心率锁在 130-160 bpm 的代偿性心动过速高原。
  * **终末休克慢阻骤降**：当失血度突破 0.80 进入终末失代偿期，心肌因极限缺氧无法维持泵血，心率发生慢阻骤跌，快速滑落至 30 bpm 甚至骤停。
* **极高拟真电生理波形合成 (P-QRS-T)**：
  * 完美重塑了标准的 Lead-II P-QRS-T 临床心电图扫描曲线。
  * **超级硬核病理形态改变 (ST Depression & T-Wave Inversion)**：当小人脑部急性缺氧（`CerebralHypoxia`）或极限心动过速（`bpm > 140`）时，心电图曲线会自动触发医学形态改变——**ST段压低** 并且 **T波对称性倒置**！玩家甚至不需要看数值，仅凭心电图上倒立的 T 波就能瞬间判定伤员正处于心肌缺氧/脑缺血极危期！
* **心室颤动 (V-Fib) 混沌波形**：
  * 心肌梗死发生时，波形不再显示整齐对称的正弦波，而是由 `37Hz/79Hz/131Hz` 三个质数频率的谐波进行非线性叠加，并配合 `7.5Hz` 的缓慢低频振幅调制。完美呈现监护仪上令人窒息的、振幅忽大忽小、宽窄不一的**杂乱混沌蠕动波**。
* **停搏平直电磁底噪与数字直观归零**：
  * 心跳骤停（Flatline）时不再是一根死板的几何死线，而是融入了 `1.8Hz` 极其缓慢的基线游走（模拟胸腔微弱游离或电磁游走）和高频底噪抖动。
  * 当心搏骤停时，心率和血氧数值显示不再使用生硬的 `---` / `--`，而是直观归零显示为 **`0`** 与 **`0%`**。

### 2. 全新临床大重命名与梗死死因解耦 (Systemic Death Chain)
* **室颤 $\rightarrow$ 心肌梗死**：将 `VentricularFibrillation` 状态完全重命名为 **“心肌梗死”**（交互文本、受伤飘字、体征仪悬停诊断均同步改写为“急性心肌梗死”）。
* **动脉破裂 $\rightarrow$ 大出血**：将 `ArterialRupture` 伤口及其 labelNoun 重命名为 **“大出血”** 与 **“大出血的伤口”**（包扎动作、医生缝合飘字、受伤弹出飘字同步汉化为“大出血”）。
* **解耦强制致死（死于拟真的多脏器衰竭自然死亡链路）**：
  * 彻底删除了心肌梗死（室颤）Hediff 节点中的 `<lethalSeverity>1.0</lethalSeverity>` 强制死亡线。进度满 1.0 时不会被引擎强制抹杀。
  * **死因生理链路**：心梗满度时会将伤员的泵血、呼吸和意识严格封锁在危急的 `1% (0.01)`。极低的心肺循环功能会瞬间激活缺氧监控，使脑缺氧以极速累积。小人将自然死于因急性心梗引起的**“心泵衰竭 -> 全身极度低灌注 -> 严重脑死亡与全身代谢性酸中毒”**的多脏器崩溃自然致死链路。

### 3. 代谢性酸中毒 (Metabolic Acidosis) Class I-IV 级增速病理重构
* **彻底移除敏感流血率恶化**：废除了原本轻微流血（`bleedRate > 0.15`）就会不断恶化酸中毒的不合理设定，将其触发条件精细化微调为：累积失血量突破 Class II 级休克门槛（>15%），或心肺维生器官衰竭度降至 <= 0.50。
* **非线性 Class I-IV 级失血性休克增速模型**：
  * **代偿期（Class I & II，失血 <= 30%）**：大幅降低了前期的恶化基数（`severityFactor = 0.18f`）。在 15% 到 30% 失血量之间，酸中毒的严重度增加极其微弱，给玩家留出极为充裕的战场就地包扎、止血带结扎与补液输血准备时间，完全杜绝了前期流血快速酸中毒的情况。
  * **代偿不足期（Class III，失血 30% - 40%）**：增长速度中度加速。
  * **极限失代偿期（Class IV，失血 > 40%）**：一旦跨入最严重的 Class IV 级濒死期（血液流失达 40% 以上），酸中毒严重度恶化速率发生**爆发性飙升（增长十六倍）**，重现晚期微循环瘫痪的死亡崩溃。

### 4. ECG 曲线丝滑连接扫描头渐变淡出算法
* 彻底废除了扫描线前方的硬性 continue 像素剔除。
* 重构为在扫描头前方的 **`8 像素`** 范围内亮芯与外辉光图层透明度的 **Alpha 线性衰减淡出**。这使得心电图折线在低分辨率、视网膜等各种屏幕缩放比率下，**呈现 100% 绝对连贯、平滑且没有任何视觉硬“断口”的极佳物理滑行扫描质感**。

### 5. 战地就地抢救与野外速度惩罚
* **病床保护**：伤员躺在任何床上（普通或医疗床，即 `patient.CurrentBed() != null`）都可以享受全速的医疗处理。
* **野外就地抢救慢速惩罚**：若完全躺在地上（野外无床就地包扎），则承担 **2.5 倍**的读条工作量慢速惩罚，体现战地急救环境的恶劣与艰难感。

### 6. 科学化血原质包 (Hemogen Pack) 战地输血
* 支持对失血伤员强行在野外进行血原质包输血以补充血容量。
* **精准失血量恢复**：摒弃原本简单粗暴的定额严重度恢复。改为动态读取伤员的致死失血上限（在您的 Mod 中已 Patch 调整为更硬核的 `0.55`），并按现实中一包血占人体总血量比例进行精准结算——每次输血可减少伤员 **12%** 的全身失血严重度（约 `0.066`）。伤员流失 2500ml 血液时，需要约 **8 包**血原质包才能完全充满。

### 7. 大出血包扎药品连续使用特权限制
* 大出血（动脉破裂）需要多次缝合包扎，为了保护医学逻辑与游戏平衡，**限制多次缝合“不消耗药品”的特权**仅限以下 5 种核心医疗品：
  * 原版三种医药：草药、工业医药、闪耀世界医药。
  * 本 Mod 专属急救包：草药急救包、战地急救箱。
* 使用其他任何第三方 Mod 药品缝合大出血时，每次包扎都将**实打实地强制扣除 1 个药品**，避免超模 Mod 药水投机取巧。

### 8. 急救对象拓宽至己方血肉生物（动物急救）
* 将原本仅限人类的右键就地急救机制拓宽至**所有血肉生物与有血生物**。
* 现在，施救医生可以随身携带急救箱或止血带，直接对己方受伤倒地的哈士奇、战马、大象等血肉宠兽或战兽在野外进行现场压迫止血和动脉缝合！

### 9. 右键急救菜单视觉完全去违和化
* **彻底移除文本前缀**：完全清除了原右键急救菜单中诸如 `[背包喂药]`、`[战地急救]` 等违和的中括号文字前缀。
* **极简化中文选项**：
  * 如果是止血操作（使用止血带，或者使用急救包/医药且小人正处于流血状态），选项文字简化为：`为 {xx} 紧急止血 (剩余: {数量})`。
  * 如果是使用其他非止血道具（如成瘾品、食物或非流血状态下使用医药），选项文字简化为：`对 {xx} 使用 {xx} (剩余: {数量})`。
* **3D 原生道具图标渲染**：利用反射分析并定位到了 `FloatMenuOption` 极其出色的原生公共字段 `iconThing`。将 `option.iconThing = firstThing;` 传入，自动在选项最左侧绘制对应急救品/医药/血包的 3D 原生渲染图标，与游戏原版右键菜单完美契合。

## 安装与配置

1. 将本 Mod 文件夹放入您的 RimWorld `Mods` 目录下。
2. 在游戏启动菜单中激活。
3. 可以在 **选项 -> 运行设置 (Mod Settings) -> Emergency Expanded** 中对大出血发生率、致命心肺阈值等进行精细数值调节，以匹配您当前的 Mod 环境。
