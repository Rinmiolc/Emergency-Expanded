# Emergency Expanded (急救与医疗扩展) - RimWorld 1.6 Hardcore Medical Mod

`Emergency Expanded` 是一款为 **RimWorld 1.6** 打造的硬核医疗与战地急救扩展 Mod。它在兼顾原版可玩性与兼容性的同时，引入了一套自洽、科学的急症病理与战地施救系统。

---

## 核心功能

### 1. 生命体征与心电图监视仪 (ECG Monitor)
在玩家选中的血肉生物控制面板中，新增全对齐的液晶体征监视仪：
* **原生 UI 对齐设计**：黑色液晶内屏采用嵌入式物理嵌合，与旁边所有正方形 Gizmo 按钮在高度、斜切角和边框外观上完美对齐。
* **100% 连续无断口折线**：采用基于扫描头前方 8 像素的 **Alpha 渐变淡出算法**，彻底消除在任何低分辨率或 UI 缩放下的像素剔除断层，保证折线丝滑连续。
* **双重代偿心率模型**：
  * 心率融合当前流血速率（BleedRateTotal）与失血严重度（BloodLoss.Severity）。
  * 急性大出血时触发交感神经代偿，心率立刻飙升 `20-60 bpm`。
  * 严重失血高原期心率维持在 130-160 bpm；终末休克期（失血度 > 0.80）代偿崩溃，心率慢阻骤降。
  * 心跳骤停（Flatline）时心率与血氧数值直接显示为归零数字 **`0`** 与 **`0%`**，波形呈现微弱电磁噪声基线与 1.8Hz 极慢胸腔起伏。
* **病理波形动态显形**：
  * **心肌缺氧/心动过速 (HR > 140)**：波形自动发生 ST段压低 (ST Depression) 与 T波对称性倒置 (T Wave Inversion) 的形态改变。
  * **心肌梗死**：呈现由非线性多谐波（37Hz/79Hz/131Hz）与低频振幅调制（7.5Hz）组成的混沌室颤蠕动波。

### 2. 代谢性酸中毒 (Metabolic Acidosis) 增速模型
* **触发判定**：代谢性酸中毒由失血量严重度（失血 > 15%）或心肺维生器官衰竭（<= 50%）触发。
* **Class I-IV 级休克非线性增速**：
  * 前期代偿阶段（失血 <= 30%）酸中毒恶化基数降至 `0.18`，进展极度缓慢，为医生留出充裕的就地急救时间。
  * 晚期第四级极危休克阶段（失血 > 40%）恶化速率呈十六倍爆发飙升，模拟休克失代偿崩盘。

### 3. 心肌梗死与大出血机制（死因解耦）
* **病理汉化重塑**：原版室颤重命名为**“心肌梗死”**，原版动脉破裂重命名为**“大出血”**（受伤飘字、包扎动作、设置菜单和体征仪诊断均已同步重塑）。
* **拟真器官衰竭致死链**：取消心梗严重度满 1.0 时的引擎强制致死判定。心梗满度时纯粹由 stages 将泵血和呼吸降为 1%，引发机体脑窒息与严重酸中毒，伤员死于拟真的全身多脏器衰竭。

### 4. 战地施救与野外惩罚
* **野外就地抢救惩罚**：伤员完全躺在地上（野外无床就地包扎）时，承担 **2.5 倍**的读条工作量慢速惩罚；在任何床上（普通床或医疗床）则享受正常处理速度。
* **血肉动物急救**：机制全面放宽至所有玩家己方的肉体生物。施救医生可直接在野外对受伤倒地的哈士奇、宠物和战兽进行包扎、压迫止血和动脉缝合。

### 5. 输血补液与耗材特权限制
* **战地精准输血**：支持在野外使用血原质包（Hemogen Pack）补充血容量。每包血原质按现实中血液容量比例精准扣减 12% 全身失血严重度。
* **药品连续缝合特权限制**：大出血（动脉破裂）需要多次缝合包扎，多次缝合不消耗药品的特权仅限官方三种医药及 Mod 专属两种急救包（草药急救包、战地急救箱）；使用其他第三方 Mod 药品缝合大出血时，每次包扎均强制消耗 1 个药品。
* **右键菜单去违和化**：彻底移除违和的中括号文本前缀，文字精简洗练（“为 xx 紧急止血” / “对 xx 使用 xx”），最左侧反射渲染原生 3D 物理道具图标。
