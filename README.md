# Emergency Expanded - 急救扩展 
### A Hardcore Medical and Emergency Expansion Mod for RimWorld 1.6

Emergency Expanded (简称EE) 是一款为 **RimWorld 1.6** 打造的硬核医疗与战地急救扩展 Mod。它在兼顾原版可玩性与兼容性的同时，引入了一套自洽、科学的急症病理与战地施救系统。
“在边缘世界，真正的致命伤从不是瞬间的爆头，而是生理机能崩溃时那令人窒息的连锁反应。Emergency Expanded 是一款致力于将 RimWorld 医疗系统推向极致拟真与硬核博弈的大修模组。我们的核心宗旨是彻底废除原版‘属性归零即暴毙’的粗暴逻辑，将生与死的界限拉长为一场争分夺秒的医疗拉锯战！在这里，您将面对深度还原的医学病理：从大出血与败血症引发的休克，到组织缺血引发的代谢性酸中毒，直至最终致命的致死三联征和多器官功能衰竭 (MODS)。模组加入了动态升阶的真实烧伤分级、残酷的骨折内固定与石膏系统、以及致命的张力性气胸。哪怕小人的心脏被一枪打碎，他们也不会瞬间死亡，而是会陷入脑缺氧的最后黄金挣扎期——只要您的军医能顶着战火进行CPR心肺复苏、静脉推注肾上腺素并紧急换心，就能创造起死回生的医学奇迹！
